A Streamlit-based chatbot app for mental well-being.  
Features include:
- Chatbot for supportive conversations
- Guided meditation links
- Daily journaling
- Mood tracker
- Motivational quotes
- Breathing exercises
- Crisis helpline info

## How to run

1. Install requirements:  
   `pip install -r requirements.txt`
2. Start the Flask backend.
3. Run the Streamlit app:  
   `streamlit run app.py`
